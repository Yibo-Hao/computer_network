# 物理层

# 物理层的目的

物理层解决如何在链接各种计算机的传输媒体上传输数据比特流，而不是指具体的传输媒体，但传输媒体本身并不属于物理层。

光纤，双绞线等都是传输媒体，物理层的作用就是尽可能地屏蔽掉传输媒体和通信手段的差异，使上面的数据链路层感受不到这些差异。

# 物理层主要任务

物理层的主要作用：确定与传输媒体接口有关的一些特性。

  • 机械特性：接口所用接线器的规格，形状，大小，引线数目。
  • 电气特性：指明接口电缆的各条线上的电压范围，-5v到+5v
  • 功能特性：规定-5v为0，+5v为1。
  • 过程特性：连接各个相关部件的工作步骤。

# 数据通信的基础知识

  • 数字信号（离散信号）：0,1，代表信息的参数是离散的，代表不同离散数值的波形就称为码元。
  • 不一定一个码元表示一个二进制数，当电气范围加大时，一个码元有可能表示三个二进制数。
  • 模拟信号（连续信号）：各种正弦波，余弦波叠加，代表信息的参数的取值是连续的。
  • 通信的目的是传送消息。
  • 消息：就是一段内容，可以是今天的天气预报
  • 数据（data）：消息在计算机内的表达
  • 信号（signal）：数据的电气和电磁的表现

信道就是向一个方向发信息的媒体。单向信道，双向交替信道，双向同时信道。

# 调制

  • 基带信号(即基本频带信号)一来自信源的信号。像计算机输出的代表各种文字或图像文件的数据信号都属于基带信号。基带信号就是发出的直接表达了要传输的信息的信号，比如我们说话的声波就是基带信号
  • 带通信号把基带信号经过载波调制后，把信号的频率范围搬移到较高的频段以便在信道中传输(即仅在一段频率范围内能够通过信道)。
  • 传输距离较近时，计算机网络都采用基带传输方式由于在近距离范围内基带信号的衰减不大，从而信号内容不会发生变化。因此在传输距离较近时，计算机网络都采用基带传输方式。如从计算机到监视器、打印机等外设的信号就是基带传输的。

# 调制的种类

调制分为两种基带调制和带通调制，基带调制后仍是基带信号，我们称之为编码（coding）把一种数字信号编码成另一种数字信号。

另一种调制称为带通调制，使用载波进行调制，把数字信号转换为模拟信号。

基带调制：不归零，归零，曼切斯特，差分曼切斯特。

带通调制：调幅，调频，调相。

# 物理层下的传输介质

  • 导引型传输媒体，非导引型传输媒体
  • 导引型传输媒体，电磁波沿着固体媒体传播，双绞线，同轴电缆，光缆
  • 非导引型传输媒体指自由空间，电磁波传输被称为无线传输，分为微波和短波。

# 物理层设备

  • 集线器：信号放大和重发，扩大网络的传输范围，不具备信号的定向传送能力，集线器会受到任何一台计算机发出的信号然后广播给其他计算机，其他计算机接不接受取决于信号上的mac地址，而不是取决于集线器发不发送，简言之一台集线器上的计算机发送数据其他计算机也会接受到。
  • 集线器是一个大的冲突域，集线器只能同时两台计算机通信其他计算机得等，集线器没有任何判断能力，相当于网线，看不懂任何信息，单纯地发送信号
  • 集线器不安全，其他计算机也能收到信号，集线器是一个冲突域需要等待其他计算机通信完，集线器均摊带宽网速很慢，集线器已经淘汰，变成交换机了。

# 信道复用技术

  • 频分复用技术：多个信号使用不同的频率来调制，到达目的地后使用不同频率解调（模拟信号）
  • 时分复用技术：一个信道上的不同计算机轮流发送信息
  • 统计时分复用：时分复用技术有的信道空时仍会占用信道，在每个数据上加上标记就是统计时分

# 数字传输系统

数字传输系统指广域网之间的通信，它用到了信道复用技术。对波形进行采集，8000/s，每一个码元代表八个二进制数，之后根据不同的标准来进行时分复用，30路或24路。8000830 = 2.048m/s。

数字电路在传输质量和经济上都有优势，所以长途都使用时分数字传输，现在模拟线路基本只剩下用户电话机到市电话机之间的线路上。

# 宽带接入技术

ADSL拨号上网，使用数字技术对模拟电话线进行改造，它把低频端留给用户打电话，高频段用来上网。使用频分复用每4khz一个信道并行传递，会有一个并传行转化器。 HFC光纤同轴混合网，使用光纤和同轴电缆结合 FTTH光纤到户，只使用光纤
